# 简介

`mangoo` 是一个“轻量异步护栏层”。

它不替代 `async/await`，而是给你补齐以下能力：
- 任务状态管理（`idle/running/success/error/aborted`）
- 取消语义（`AbortSignal`）
- 并发控制（限流、失败策略）
- React / Vue 一致的 Hook 心智

## 适用场景

- 表单提交流程：保存、校验、写审计日志
- 页面初始化：多个接口并发加载
- 可取消操作：切路由、切条件时中断旧请求
- 需要统一错误对象和状态上报的业务

## 不适用场景

- 你只需要一次性 `await fetch()`，没有状态/取消/并发诉求
- 你已经有完整状态机系统，且团队不希望引入另一层抽象
